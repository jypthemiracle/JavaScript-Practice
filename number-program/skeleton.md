**숫자 야구게임 - node.js로 구현**

1. 컴퓨터(상대방) 구현

* 컴퓨터가 임의의 숫자 3개를 선택하는 메소드

2. 사용자 구현

* 사용자가 숫자를 입력하는 메소드

3. 심판 역할 구현

* 게임 진행상황을 판단하는 메소드
  * 몇 번째 시도인지 출력
  * 현재까지 맞춘 숫자를 출력
  * 맞춘 시간을 출력

* 컴퓨터의 숫자와 사용자의 숫자를 비교하는 메소드
  * 같은 수가 있는가?
    * 같은 수가 같은 자리에 있는가?
    * 같은 수가 다른 자리에 있는가?
  * 같은 수가 없는가?
* 결과값을 판정하여 출력하는 메소드
  * 위의 메소드에서 결과값을 받아와 스트라이크 / 볼 / 포볼이라는 판정을 내림
